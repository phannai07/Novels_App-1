# Scope
_ user can login and sign up 
_ user can view all books with details (title, author, price, cover, discription)
_ user can Search books by name, add books to shopping cart, and purchase books
_ user can add place order 
- user can View order history
- They can change light and dark mode 
# Functionalities
1. Home:
   - Display all books (title,price, and cover image),upcoming , recomentdation , and popular.
   - Search bar: search by title
3. Menu:
   - Display types of book
4. Book detail:
   - Title, author, price, cover, pages, discription
   - Buttons: Add to Cart
5. Shopping cart :
   - button Add/remove books , Update quantity, Show total price.
6. Payment
   - pay by QR code
7. Sitting:
   - user profile: name, email, password
   - Dark and light mode
   - History : date, tittle , total price, and payment
   - Logout
