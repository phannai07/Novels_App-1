# Novels_App
# Introduction
The Novels_App is a mobile app that works like an online bookstore. Users can sign up, log in, and look at many novels. They can see details about each book like the title, author, price, and description. The app also lets users search for books, add them to a shopping cart, and buy them safely. In addition, user can check their order history and location of store.
# ProblemStatement
Many readers find it hard to buy novels because bookstores are not always near them, and some books are not available in local shops. It also takes time to search for the right book in physical stores. People want an easier way to find, read details, and buy books anytime.

The Novels_App solves this problem by giving users a mobile app where they can quickly search, view, and purchase novels without going to a physical store.
# Scope
- user can login and sign up
- user can view all books with details (title, author, price, cover, discription)
- user can Search books by name, add books to shopping cart, and purchase books
- user can add place order 
- user can View order history
